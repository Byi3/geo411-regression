Title - Utilizing Regression to Analyze SAT Scores and Predicted GPA and Amount of Mosquito Development Sites and Predicted Pupae Populations

Overview - Using R to analyze level of regression in regression models made for the aforementioned research topics.

Methods - Used R commands to generate histograms, scatterplots, and regression models.

Key Findings - 79.14% of GPA could be explained by SAT scores, and about 70.09% of pupae populations could be explained by mosquito development sites.

Tools - R Studio

Reflection - R Studio is apparently capable of generating graphs that remove outliers, but can still highlight where the outliers were in the previous graphs which is neat. Of course, R Studio is capable of doing regression calculations which is very handy.
